## Microbial Genetic Algorithm
微遗传算法利用了微生物的对抗和感染，微遗传算法每一次随机选取种群中的二者进行对抗，选出赢家(winner)和输家(loser)，将赢家的一部分基因感染至输家，将二者在回归种群完成一次进化。

## Group Microbial Genetic Algorithm
将随机选择种群的二者改为将种群分成两组，每组按顺序各选择一个进行对抗，将输家突变回归种群。
